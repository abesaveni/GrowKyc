## 🎨 **DESIGN SYSTEM**

### **Color Palette** (Xero-inspired)

**Primary**:
- **Cyan**: `#13B5EA` - Primary actions, links
- **Dark Blue**: `#0E7C9E` - Hover states, accents

**Semantic**:
- **Success Green**: `#3DD598` - Pass, complete, success
- **Accent Orange**: `#FFA300` - Warnings, attention
- **Error Red**: `#EF4444` - Errors, failed, critical

**Neutral**:
- **Gray 50**: `#F9FAFB` - Backgrounds
- **Gray 100**: `#F3F4F6` - Hover states
- **Gray 900**: `#111827` - Text

### **Typography**

**Font Family**: Inter (modern, readable)

**Sizes**:
- **3xl**: Headlines (30px)
- **2xl**: Subheadlines (24px)
- **xl**: Section titles (20px)
- **lg**: Card titles (18px)
- **base**: Body text (16px)
- **sm**: Captions (14px)
- **xs**: Meta info (12px)

### **Spacing Scale**

Consistent 4px base unit:
- **1**: 4px
- **2**: 8px
- **3**: 12px
- **4**: 16px
- **6**: 24px
- **8**: 32px
- **12**: 48px

### **Components**

All components use:
- Rounded corners (8-16px)
- Subtle shadows
- Hover states
- Transition animations
- Consistent padding

## ✅ **HOW TO USE**

### **1. Replace Current Module**

In `/src/app/App.tsx`, replace:

```tsx
import { GrowKYC } from './components/grow-kyc/GrowKYC';
```

With:

```tsx
import { GrowKYCRedesigned } from './components/grow-kyc/GrowKYCRedesigned';
```

And update the render:

```tsx
if (currentModule === 'grow_kyc') {
  return <GrowKYCRedesigned />;
}
```

### **2. Test the New Experience**

- Navigate to Grow KYC module
- Try keyboard shortcuts (Cmd+K, Cmd+N, ?)
- Click Quick Add to start onboarding
- Drag & drop some files
- Complete the tour

### **3. Gather Feedback**

- Ask users to complete common tasks
- Track time to completion
- Measure satisfaction
- Collect feature requests
